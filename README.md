WYYCMS

WYYCMS 基于 BEEGO开发，能够以最低的成本、最少的人力投入在最短的时间内架设一个功能齐全、性能优异、易于维护的管理后台。

wyycms 主要功能：

网站管理 产品管理 文章管理 系统管理 后台权限功能 菜单自定义、权限自定义

前台地址 http://www.weiyeying.cn 

后台地址 http://www.weiyeying.cn/admin
